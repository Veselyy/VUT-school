# Web — online verze příprav

Tato složka se **generuje při buildu**. Needituj ji ručně.

| Cesta | Předmět |
|-------|---------|
| `web/ipp/` | IPP |
| `web/izg/` | IZG |

Lokální náhled: otevři `IPP/otazky.html` nebo `web/ipp/index.html`.

## GitHub Pages

GitHub vyžaduje složku `docs/`. Z kořene repa:

```bash
make pages-sync
```

Pak Settings → Pages → `/docs`.
